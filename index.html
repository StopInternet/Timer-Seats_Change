ランダム席替え

カウントダウン形式のタイマーと エクセルで入出力ができるランダム席替えが使えるサイトです。

使う際はタブから選択してください。

タイマーの説明

時間の設定

Set Time

カウントダウン形式の時間が設定できます。
入力は分単位でお願いします。
最大で1000分間の計測が可能となっています。
1000分超える入力をした場合は自動的に1000分に設定されます。

アラートの設定

SET ALERT

途中で鳴らす経過音を設定できます。
最大3回まで設定できます。

例：
時間設定:50分
アラートの設定:10分
この場合、残り10分になった時に経過音がなります。






ランダム席替えの説明

席数の設定

Set Seats

縦と横の席数を設定することができます
最大縦9、横9の81席まで設定ができます。

席を作る

CREATE SEATS MODEL

席数を設定した状態でクリックすると番号で設定された席一覧を見ることができます。

ランダムシャッフル

RANDOM SHUFFLE

席数の設定と席を作るを終えた状態でクリックすると、ランダムに席替えを行います。
クリックするたびに席が入れ替わります。

名前の入力欄

NAME input field

席数の設定と席を作るを終えると席数分、名前の入力欄が生成されます。
また、検索欄から番号を入力するとその番号が一番上に表示されます。

エクセルへの入出力

Input/output to/from Excel

1.名前入力欄を出力（Excel）
名前の入力欄で名前を入力した後に、
クリックをすると Excelファイルに入力したデータを出力をします。

2.名前入力欄を読みこむ。（Excel）
上記の1を終えた後にクリックして出力したファイルを読み込むことで、
出力したデータが自動的に入力されます。

3.席替え後のデータを出力（Excel）
ランダムシャッフルを終えた後にクリックすると、
席替え後のデータをExcelファイルに出力ができます
注意:Excelファイルの中の上記に数字がでるため数字がある部分は消してください。
